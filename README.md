# UselessApp

A Swift Mac application template for creating hilariously pointless applications.

## Requirements
- macOS 14.0+
- Xcode 15.0+
- Swift 5.0+

## Getting Started
1. Open `UselessApp.xcodeproj` in Xcode
2. Select your Mac as the target device
3. Press Cmd+R to build and run
4. Start implementing useless features from `CLAUDE.md`

## Project Structure
- `UselessAppApp.swift` - App entry point with window configuration
- `ContentView.swift` - Main view where you build your useless UI
- `Assets.xcassets` - Images and color assets
- `UselessApp.entitlements` - App sandbox permissions

## Building
```bash
xcodebuild -project UselessApp.xcodeproj -scheme UselessApp -configuration Release
```
